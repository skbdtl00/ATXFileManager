# ATX File Manager - User Guide

Welcome to ATX File Manager! This guide will help you get started with using the file manager.

## Getting Started

### Creating an Account

1. Navigate to the registration page
2. Enter your email, username, and password
3. Click "Register"
4. Check your email for verification (if enabled)

### Logging In

1. Go to the login page
2. Enter your email and password
3. If you have two-factor authentication enabled, enter the code from your authenticator app
4. Click "Login"

## Basic File Operations

### Uploading Files

1. Click the "Upload" button
2. Select files from your computer
3. Choose the destination folder (optional)
4. Files will be uploaded and appear in your file list

**Supported Features:**
- Drag and drop upload
- Multiple file selection
- Progress indicator
- Automatic thumbnail generation for images

### Creating Folders

1. Click "New Folder" button
2. Enter a folder name
3. Click "Create"

### Downloading Files

1. Select a file
2. Click the download icon or right-click and select "Download"
3. The file will be downloaded to your computer

### Renaming Files

1. Right-click on a file or folder
2. Select "Rename"
3. Enter the new name
4. Press Enter or click outside to save

### Moving Files

**Method 1: Drag and Drop**
1. Click and hold on a file
2. Drag it to the destination folder
3. Release to move

**Method 2: Cut and Paste**
1. Right-click on a file
2. Select "Cut"
3. Navigate to destination folder
4. Right-click and select "Paste"

### Copying Files

1. Right-click on a file
2. Select "Copy"
3. Navigate to destination folder
4. Right-click and select "Paste"

### Deleting Files

1. Select files or folders
2. Press Delete key or click the delete button
3. Confirm the deletion

**Note:** Deleted files go to trash by default and can be recovered. To permanently delete, hold Shift while deleting.

## Advanced Features

### Search

1. Click the search bar at the top
2. Type your search query
3. Results appear in real-time
4. Use filters to narrow results:
   - File type
   - Date range
   - Size range
   - Tags

### Starring Files

Mark important files with a star:
1. Click the star icon on any file
2. Starred files appear in the "Starred" section for quick access

### File Tags

Organize files with tags:
1. Right-click on a file
2. Select "Add Tag"
3. Enter tag names (comma-separated)
4. Tags appear below the filename

**Using Tags:**
- Click a tag to see all files with that tag
- Search for tags using `tag:name` in search

### File Preview

Supported file types can be previewed:
- **Images**: Full-size preview with zoom
- **Documents**: PDF viewer
- **Text/Code**: Syntax-highlighted editor
- **Videos**: Built-in player

To preview:
1. Click on a file
2. Preview appears in the right panel
3. Use toolbar for additional options

### Sharing Files

Create share links for files:
1. Right-click on a file
2. Select "Share"
3. Configure share settings:
   - Expiration date
   - Password protection
   - Download limit
   - IP restrictions
4. Copy the share link

**Share Link Features:**
- Public access without login
- Track download count
- Revoke access anytime
- Set expiration dates

### Archives

**Creating Archives:**
1. Select files or folders
2. Right-click and select "Create Archive"
3. Choose format (ZIP or TAR)
4. Enter archive name
5. Archive is created in current folder

**Extracting Archives:**
1. Right-click on an archive file
2. Select "Extract"
3. Choose destination folder
4. Files are extracted

Supported formats:
- ZIP
- TAR/TAR.GZ
- 7Z (if installed)

### File Information

View detailed file information:
1. Right-click on a file
2. Select "Properties"

Information includes:
- File size
- Type and mime type
- Created/modified dates
- File hashes (MD5, SHA256)
- Storage location
- Tags
- Share links

## View Modes

Switch between different view modes:

**Grid View:**
- Visual thumbnails
- Good for images and media
- Shows file icons

**List View:**
- Detailed file information
- Sort by columns
- Compact display

**Toggle views** using the view buttons in the toolbar.

## Keyboard Shortcuts

- `Ctrl+A` - Select all
- `Ctrl+C` - Copy
- `Ctrl+X` - Cut
- `Ctrl+V` - Paste
- `Delete` - Delete (move to trash)
- `Shift+Delete` - Permanent delete
- `F2` - Rename
- `Ctrl+F` - Search
- `Ctrl+Z` - Undo
- `Escape` - Cancel/Close
- `Space` - Preview file

## Storage Management

### Checking Storage Usage

1. Click on your profile icon
2. Select "Storage"
3. View storage usage breakdown by:
   - File types
   - Folders
   - Time period

### Managing Storage

When approaching quota:
1. Delete unnecessary files
2. Clean up trash
3. Remove old versions
4. Compress large folders

### Storage Quota

- Free tier: 10 GB
- Upgrade available in settings
- Quota includes all files and versions

## Two-Factor Authentication

Secure your account with 2FA:

**Setup:**
1. Go to Settings ‚Üí Security
2. Click "Enable Two-Factor Authentication"
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

**Login with 2FA:**
1. Enter email and password
2. Enter 6-digit code from authenticator app
3. Login completes

## Troubleshooting

### Upload Issues

**File too large:**
- Check file size limits in settings
- Compress files before uploading
- Contact admin for increased limits

**Upload fails:**
- Check internet connection
- Ensure sufficient storage quota
- Try smaller batches of files

### Cannot Access Files

- Check if file was moved or deleted
- Verify permissions
- Check if sharing link expired

### Slow Performance

- Clear browser cache
- Close unused tabs
- Check internet connection
- Use list view instead of grid view

## Tips and Tricks

1. **Batch Operations:** Select multiple files with Ctrl+Click or Shift+Click
2. **Quick Upload:** Drag files directly onto the browser window
3. **Navigation:** Use breadcrumb trail to jump to parent folders quickly
4. **Recent Files:** Access recently modified files from the sidebar
5. **Dark Mode:** Toggle in settings for comfortable viewing at night
